# 验证证据

## 自动测试

当前 52 项确定性测试覆盖：

- 稳定哈希边界、权重、虚拟节点与环查询；
- Rendezvous 排名、盐命名空间和权重分布；
- 区域/机架隔离及约束不足时的明确降级；
- 节点加入、下线、主副本变化和移动比例；
- 分布偏斜、不完整放置与拓扑校验；
- 五阶段迁移顺序、波次上限和单节点压力预算；
- 无可读源阻塞、计划确定性与 JSON 输出。

验证命令：

```bash
moon fmt --check
moon check --target all
moon test --target js
moon test --target wasm
moon test --target wasm-gc
```

## 固定工作负载

`moon run bench/main --target js` 对 10,000 个固定键执行三副本放置、八到九
节点扩容分析和安全迁移编排，输出：

- 主分片负载范围与偏斜；
- 移动键数和移动比例；
- 迁移动作与波次数；
- 阻塞键和计划校验问题数；
- 输入结果摘要哈希。

输出是实际运行证据，不宣称未经统一硬件计时验证的吞吐率。
