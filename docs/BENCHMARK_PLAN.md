# 性能与质量验证计划

## 数据集

- 1k、10k、100k 和 1m 个固定键；
- 4、8、16、32、64 个节点；
- 权重 1:1、1:2、1:4 和混合权重；
- 1、2、3、5 个副本；
- 单节点加入、退出、变权和整区不可用。

## 指标

- 每秒主节点查询数；
- 三副本放置耗时；
- 环构建时间与虚拟点内存；
- 扩缩容移动比例；
- 最大/最小主负载及相对偏斜；
- 区域和机架约束满足率。

## 复杂度

- 环构建：O(V log V)，V 为总虚拟点数；
- 环查询：O(log V)，副本扫描取决于相邻虚拟点；
- Rendezvous：O(N × W)，N 为节点数，W 为票据权重；
- 迁移计划：O(K × placement)，K 为样本键数；
- 分布分析：O(K × placement)。
