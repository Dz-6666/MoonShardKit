# 项目价值说明

## 面向场景

- 分布式缓存与对象存储；
- 消息分区和任务队列；
- 搜索索引、时序数据与租户路由；
- 游戏房间和会话归属；
- 边缘节点、Wasm 服务和本地仿真。

## 不只是哈希函数

哈希函数只把输入映射为整数，不能回答节点权重、副本隔离、扩缩容移动量和负载
偏斜。MoonShardKit 把这些运维问题建模为可重复计算的纯算法：

- 放置结果可以在上线前离线审计；
- 拓扑变化可以先生成迁移计划再执行；
- 约束不足不会静默失败；
- 所有结果可以进入 CI 做回归比较。

## 后端中立

核心不读取时钟、不使用随机数、不连接注册中心，也不依赖线程和网络。相同输入
可在浏览器 Wasm、边缘运行时、Node.js 和 Native 控制面复用。
